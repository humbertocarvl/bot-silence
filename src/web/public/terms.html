Termos de Serviço

Silence Bot — Última atualização: 15 de maio de 2026

1. Aceitação dos Termos

Ao utilizar o Silence Bot ("Bot") no servidor do Discord Silence, você concorda com estes Termos de Serviço. Caso não concorde, não utilize as funcionalidades do Bot.

2. Descrição do Serviço

O Silence Bot é um bot para Discord desenvolvido para o servidor da comunidade Silence. Suas funcionalidades incluem:

  • Gerenciamento automático de cargos no processo de entrada de novos membros
  • Verificação de contas do Habbo Hotel, vinculando-as ao perfil do Discord
  • Ferramentas de administração para moderadores do servidor
  • Integração com a API pública do Habbo Hotel BR

3. Uso Aceitável

Ao usar o Bot, você concorda em não:

  • Tentar verificar contas do Habbo que não sejam suas
  • Usar o Bot para spam, assédio ou qualquer atividade maliciosa
  • Tentar explorar vulnerabilidades, contornar restrições ou manipular o sistema de verificação
  • Usar automações para interagir com os comandos do Bot de forma abusiva

4. Verificação de Conta Habbo

O processo de verificação requer que você insira temporariamente um código fornecido pelo Bot no campo "Missão" do seu personagem no Habbo Hotel. Ao utilizar esse recurso:

  • Você confirma ser o legítimo proprietário da conta do Habbo informada
  • Você autoriza o Bot a consultar as informações públicas do seu perfil Habbo via API pública
  • Apenas uma conta do Habbo pode ser vinculada por usuário do Discord de cada vez

5. Disponibilidade

O Bot é oferecido sem garantias de disponibilidade contínua. Podemos interromper, modificar ou descontinuar funcionalidades a qualquer momento, com ou sem aviso prévio. Não nos responsabilizamos por perdas decorrentes de indisponibilidade.

6. Modificações

Reservamo-nos o direito de alterar estes Termos a qualquer momento. O uso continuado do Bot após alterações constitui aceitação dos novos Termos.

7. Contato

Para dúvidas ou solicitações relacionadas a estes Termos, entre em contato através do servidor do Discord ou abra uma issue no repositório do projeto.
