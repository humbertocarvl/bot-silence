Política de Privacidade

Silence Bot — Última atualização: 15 de maio de 2026

1. Introdução

Esta Política de Privacidade descreve quais dados o Silence Bot coleta, como os utiliza e como os protege. Ao utilizar o Bot, você concorda com as práticas descritas aqui.

O Silence Bot não é afiliado à Sulake Corporation nem ao Habbo Hotel. Os dados do Habbo são obtidos exclusivamente por meio da API pública disponibilizada em habbo.com.br.

2. Dados Coletados

Coletamos apenas os dados estritamente necessários para o funcionamento das funcionalidades do Bot:

Dado Origem Finalidade
ID do usuário Discord Discord API Identificar o usuário no processo de verificação
Nome de usuário do Habbo Informado pelo usuário + confirmado via API Habbo Vincular a conta Habbo ao perfil Discord
ID único do Habbo (uniqueId) API pública do Habbo Hotel BR Identificar de forma única a conta Habbo vinculada
Data e hora da verificação Gerado internamente Registro de quando a vinculação foi realizada

Não coletamos senhas, e-mails, informações financeiras nem qualquer dado sensível.

3. Como os Dados São Usados

  • Para realizar e confirmar a vinculação entre sua conta Discord e sua conta Habbo
  • Para atribuir o cargo de verificado no servidor após confirmação
  • Para exibir informações da sua conta Habbo vinculada ao usar comandos como /minhaconta
  • Para estatísticas internas do servidor (ex.: total de contas verificadas)

Os dados não são vendidos, compartilhados nem transferidos a terceiros.

4. Armazenamento e Segurança

Os dados são armazenados em um banco de dados PostgreSQL hospedado na plataforma Railway, com conexão criptografada (SSL). O acesso ao banco é restrito ao Bot.

Códigos de verificação temporários são armazenados por no máximo 60 minutos e excluídos automaticamente após o uso ou expiração.

5. Dados de Terceiros — API do Habbo

Durante a verificação, o Bot consulta a API pública do Habbo Hotel BR para confirmar a existência do usuário e verificar o campo "Missão" do personagem. Apenas dados disponíveis publicamente na API são acessados. Consulte a política de privacidade do Habbo para detalhes sobre os dados que eles disponibilizam.

6. Seus Direitos

Você pode a qualquer momento:

  • Solicitar a exclusão dos seus dados — entre em contato com a administração do servidor
  • Consultar quais dados estão associados à sua conta usando o comando /minhaconta

Após a exclusão, o cargo de verificado será removido e a vinculação desfeita.

7. Retenção de Dados

Os dados de verificação são mantidos enquanto o usuário permanecer no servidor ou até que solicite a exclusão. Dados de verificações pendentes (não concluídas) são excluídos automaticamente após 60 minutos.

8. Alterações nesta Política

Podemos atualizar esta Política periodicamente. Alterações relevantes serão comunicadas no servidor do Discord. O uso continuado do Bot após as alterações constitui aceitação da nova Política.

9. Contato

Para exercer seus direitos ou tirar dúvidas sobre privacidade, entre em contato pela administração do servidor no Discord.
